# Message Wrapper

This package just for my own tests and learning

## Install

Via Composer

``` bash
$ composer require nelex/messageWrapper
```

## Usage

``` php
$wrapper = new Nelex\MessageWrapper();
echo $wrapper->echoPhrase('Steve','Hi, ','!');
```

## Testing

``` bash
$ phpunit
```
